### Setup
run `npm i`
install the needed Dependencies `JEST, ESLINT, CORS, DOTENV, MORGAN, MONGOOSE, EXPRESS`.<br>
also you need `base-64`, `bcryptjs` and `jsonwwebtoken` for authentication/authorization sign-in by github account.
